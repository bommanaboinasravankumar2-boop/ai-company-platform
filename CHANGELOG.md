# Changelog

All notable changes to **Synapse OS** will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.1.0] - 2026-07-18
### Added
- **Interactive Audit Screenshots**: Created the `AuditScreenshot` sub-module inside the CRM Client Finder dashboard, generating visual overlays for crawled client web domains.
- **Simulated Conversion Pipeline**: Introduced a toggle switch inside the screenshot audit workspace to simulate the optimized state, embedding an active, fully functional Aura AI support chatbot and showing performance enhancements (98/100 score).
- **Responsive Navigation Flow**: Extended lead details with tab switching between Executive Summary and Interactive Audit Screenshot, enhancing outbound lead targeting speed.

### Changed
- Improved CRM listing filters and data alignment.
- Enhanced linter standards and strict TypeScript check boundaries on build workflows.

---

## [1.0.0] - 2026-06-15
### Added
- **Multi-Agent Dashboard (Synapse OS)**: Initial release of the unified business command station.
- **CRM & Client Finder**: Built-in lead tracking database with missing capability filters and automatic firmographic summaries.
- **Document & Email Autopilots**: Direct generation of customized business proposals and outreach emails using server-side Gemini API.
- **Express-Vite Container Middleware**: Full-stack architecture with standard bundle routing and standalone execution readiness.
- **Audit Logging**: Local activity trackers capturing all dashboard adjustments, database additions, and AI triggers.
